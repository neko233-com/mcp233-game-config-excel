mcp233-game-config-excel

游戏 Excel 配表 MCP CLI。严格读取和写入 config233 固定行格式；默认模板直接匹配 I18nTipsConfig。

无网络运行时本地 .xlsxJSON-RPC 2.0

检查

读取 CLIENT、TYPE、SERVER 行和数据行数量。

校验

验证固定行、字段顺序、字段类型与 UID 唯一性。

批量安全操作

递归搜索、正则、去重、格式风险检查；替换默认预览，确认后写入。

开始

# 创建 I18nTipsConfig.xlsx
mcp233-game-config-excel init-i18n --file .\I18nTipsConfig.xlsx

# 验证服务端字段和 id 唯一性
mcp233-game-config-excel validate --file .\I18nTipsConfig.xlsx --expected-columns id,tips_CN

# 新增或更新一条中文提示
mcp233-game-config-excel upsert --file .\I18nTipsConfig.xlsx --uid network_error --values '{"tips_CN":"网络异常，请重试"}'

# 新增图鉴图标列：保留相邻列样式，统一按文本保存
mcp233-game-config-excel add-column --file .\FishingWeaponConfig.xlsx --name handbookIconPath --client-name handbookIconPath --display-name "图鉴显示的武器 icon" --type string --comment "图鉴显示的武器 icon" --after-column skillId

# 校验目标列的文本格式
mcp233-game-config-excel check-column --file .\FishingWeaponConfig.xlsx --name handbookIconPath --require-text true

# 导出每个语言独立 JSON；自动识别 tips_CN / tips_EN
mcp233-game-config-excel export-i18n --file .\I18nTipsConfig.xlsx --output-dir .\i18n --format json --mode full

# 递归搜索，支持正则；返回路径、Sheet、单元格和去重值
mcp233-game-config-excel search --project-config .\mcp233-game-config-excel.txt --query 'icon_handbook_.*' --regex

# 先预览完整修改前后对比；确认后再加 --apply
mcp233-game-config-excel replace --project-config .\mcp233-game-config-excel.txt --query 'old_icon' --replacement 'new_icon'
mcp233-game-config-excel replace --project-config .\mcp233-game-config-excel.txt --query 'old_icon' --replacement 'new_icon' --apply

# 查出 string 列被 Excel 自动转日期/数字等非文本风险
mcp233-game-config-excel check-formats --project-config .\mcp233-game-config-excel.txt

# 深度自动校验。已打开 Excel 会被列出：可读不可写
mcp233-game-config-excel deep-validate --project-config .\mcp233-game-config-excel.txt

# 同一 MCP 进程内搜索缓存统计
mcp233-game-config-excel cache

# 全项目规范审计：id 唯一性、战斗原始数值 100 = 显示 1.00
mcp233-game-config-excel validate-project --project-config .\mcp233-game-config-excel.txt

# 离线自动更新：启动器自动选最新本地版本；无网络下载
mcp233-game-config-excel update-status

工具没有 HTTP 客户端、数据库连接、遥测或登录逻辑。MCP 仅通过 stdio 与宿主通信，文件路径由调用方提供。

config233 格式

行A 列B 列及之后
1注释字段说明

3CLIENT客户端字段名

config_excel_delete_column按字段名删除整列。
config_excel_check_column_format检查列位置、文本格式和异常单元格。
config_excel_search递归目录搜索；支持正则、精确位置和值去重。
config_excel_replace批量替换；默认预览每格修改前后，apply=true 才写盘。
config_excel_check_text_formats检查全部 string 字段的非文本、日期自动转换风险。
config_excel_deep_validate自动验证全部表结构、格式风险和 Excel 打开锁文件。
config_excel_cache查看或清空进程内工作簿索引缓存。

项目目录配置

项目根 mcp233-game-config-excel.txt 指定业务配表目录。支持 properties 和 TOML；相对路径基于此文件解析。

config_dirs = ["Team-Resources/BusinessConfig"]
id_column = "id"
battle_value_scale = 100
battle_attribute_patterns = ["(?i)(attr|talent|speed)"]

命中战斗字段规则的整数原始值固定按 显示值 = 原始值 / 100 解释：100 显示 1.00，235 显示 2.35。不可填写 2.35。

Excel 正打开时会出现 ~$*.xlsx 锁文件。MCP 仍可搜索和检查，但所有写操作会先整体拒绝，并明确指出必须关闭的表，防止部分写入。

搜索在同一 MCP 进程内缓存未变化工作簿；文件变更或 MCP 写入自动失效。默认最多返回 100 个位置，完整命中数和值去重仍保留。

启动器只选择本机已构建的版本化二进制，不联网下载或覆盖运行中的 exe；支持自动选择最新版本、固定版本和安全回退。

多语言全量与增量

自动识别 tips_CN、tips_EN、name_JP 与 tips_zh_CN。非标准字段可显式传入 title=CN,description=EN。

模式JSONCSV / TSV
full{ "key": "text" }key,value
incremental{ "upsert": {...}, "delete": [...] }operation,key,value

增量模式要求上一版兼容 Excel 作为基线。删除行明确输出为 delete；字符串清空输出为空值 upsert。
